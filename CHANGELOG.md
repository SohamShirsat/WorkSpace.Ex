# Changelog

All notable changes to WorkSpace are documented here.

## [1.0.0] — 2026-04-28

### Added
- Workspace management — create, edit, duplicate, delete, and reorder workspaces
- Per-workspace custom icon and color
- Launch workspace in current window or a new window
- Optional auto-close of unrelated tabs on workspace launch
- Auto-group launched tabs using Chrome Tab Groups with configurable color and name format
- Focus Mode — one-click toggle with badge indicator, optional close-others and new-window behaviour
- Duplicate tab detection with banner notification and one-click clean-up
- Tab Sequencer — reorder open tabs by domain frequency or custom regex rules
- Time Tracker — per-domain active time for today and the current week, with configurable idle threshold
- Scratchpad — temporary URL list with paste-multiple, add-current-tab, open-all, and save-as-workspace
- Session Saver — snapshot all open tabs as a new workspace instantly
- Full-text search across workspaces and open tabs
- Export / import workspaces as JSON
- Settings page with General, Focus Mode, Tab Groups, Shortcuts, Advanced, and Data sections
- System / light / dark theme support
- Keyboard shortcut to open popup (`Ctrl+Shift+W` / `Cmd+Shift+W`)
